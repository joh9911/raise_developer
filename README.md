"개발자 키우기" 게임 입니다.

게임 속에서 실제 게임 유저의 정보와 상호작용 할 수 있는 게임을 만들고 싶었습니다.

이에 기존 키우기 게임 방식에 더하여

유저의 Github Contribution 정보를 이용해 재화를 얻을 수 있는 기능이 합쳐진 게임을 만들게 되었습니다.
